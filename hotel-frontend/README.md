# Hotel Management System

A comprehensive hotel management admin dashboard built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- **Dashboard Overview**: Real-time statistics, recent bookings, top customers, and booking calendar
- **Rooms Management**: View, search, filter, and manage hotel rooms with status tracking
- **Bookings Management**: Comprehensive booking table with filters, pagination, and status management
- **Customer Database**: Track customer information, booking history, and spending
- **Billing & Invoices**: Manage payments, invoices, and revenue tracking
- **Settings**: Configure hotel information, notifications, security, and appearance

## Tech Stack

- **React 18** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **Tailwind CSS v4** - Styling
- **React Router** - Client-side routing
- **Lucide React** - Icons
